# Upload files to s3
this program will help you to upload all files to s3 which exist in current dir inside upload_folder and provide you all the public urls of files
# Requiruement
- Rust
## Add .env to current dir
```env
AWS_ACCESS_KEY_ID=
AWS_REGION=
AWS_BUCKET_NAME=
AWS_SECRET_ACCESS_KEY=
FILE_JSON=
```
